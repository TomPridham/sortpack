# sortpack

Organize your package.json

It's like Prettier for your package.json. Inpsired by fixpack, but actively maintained!

--------

## Installation

`npm i -D fixpack`

## Usage

`fixpack`

Add to your package.json:

```
"scripts": {
  "something": "sortpack"
}
```

Options:

```
-i [--indent] # one of: 2 4 tab
```
